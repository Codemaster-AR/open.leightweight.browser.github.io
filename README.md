# Open Lightweight Browser
=========================

A lightweight, fast, and feature-packed browser designed for everyday use.

### About Open Lightweight Browser

Our browser is built with simplicity and performance in mind. It is extremely light, using minimal system resources, but still provides a rich browsing experience.
